# HR-Analytics-Dashboard
# Objectives
1) Analyze employee attrition and retention patterns

2) Identify key drivers contributing to employee turnover

3) Enable data-driven HR decision-making through visual insights
# Dataset Used
- <a href="https://github.com/Mahalakshmi105hub/HR-Analytics-Dashboard/blob/main/HR%20Data.xlsx">Dataset</a>

# Dashboard Features

1) Attrition analysis by department, job role, gender, age group, and tenure

2) KPIs including attrition rate, average age, job satisfaction, and years at company

3) Identification of high-attrition and high-risk employee segments

4) Interactive slicers and filters for dynamic exploration

- Dashboard interaction <a href="https://github.com/Mahalakshmi105hub/HR-Analytics-Dashboard/blob/main/HR%20Analytics%20Dashboard.png">View Dashboard</a>

# Tools & Technologies

1) Power BI – Data modeling, DAX measures, and interactive dashboards

2) Microsoft Excel – Data cleaning, preprocessing, and transformation

3) HR Analytics – Workforce metrics, attrition analysis, trend identification

# Key Insights

- Certain departments and job roles show higher attrition trends

- Employees with lower job satisfaction and shorter tenure are more likely to leave

- Attrition patterns vary significantly across age groups and experience levels

# Outcome

The dashboard provides a comprehensive view of workforce data, enabling HR stakeholders to:

- Identify attrition risks early

- Improve employee retention strategies

- Support strategic workforce planning using analytics
